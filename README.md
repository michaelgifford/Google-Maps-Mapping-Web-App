# Google-Maps-Mapping-Web-App
Mapping web app made with Angular, MongoDB, Express, and Node. Communicates with the Google Maps API.

# Usage
1.) Download and unzip files into a folder on your system

2.) Open your terminal and go to folder with project files

3.) Run 'npm install'

4.) Launch 'mongod'

5.) Run 'node server.js' in another terminal window/tab

6.) Open a web browser and navigate to 'http://localhost:3000/'
